# Анализ лояльности пользователей Яндекс Афиши

## Описание проекта
Анализ пользовательской лояльности для сервиса Яндекс Афиша.

## Установка
1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Скопируйте `.env.example` в `.env` и заполните параметры
4. Запустите Jupyter Notebook: `jupyter notebook analysis.ipynb`

## Структура проекта
- `analysis.ipynb` - основной ноутбук с анализом
- `requirements.txt` - зависимости Python
- `.env.example` - шаблон для конфигурации
- `.gitignore` - игнорируемые файлы
- `README.md` - документация
